## 投递渠道（按推荐度）

1. **内推**（最优）：找朋友、知乎私信、LinkedIn 联系上
2. **官网投递**
3. **猎头**（Tier 2 / 3 用）
4. **BOSS 直聘**（Tier 2 / 3 用）
5. **牛客网**（实时岗位池）

## 投递前的"对齐检查"

每投一家公司前，做这件事：

```
1. 把 JD 关键词列出来（vLLM / Megatron / CUDA / RL ...）
2. 在简历最相关的项目描述里把这些关键词嵌进去
3. 写定制化 Cover Letter 一段话（你为什么对这家公司感兴趣 + 你能做什么）
```

## 面试节奏管理

- **同一时段不要超过 5 家在面**（精力顶不住）
- **优先级低的公司用来"练手"**（先投 Tier 3 一家，面完再投 Tier 1）
- **每家面完写"自我反思"**（哪个问题没答好、下次怎么补）
