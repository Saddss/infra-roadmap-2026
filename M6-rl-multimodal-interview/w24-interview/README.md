# W24：面试冲刺 + 投简历

## 目标

- 把 M1-M5 的所有产出物串成可讲的故事
- 八股 + 真题刷一遍
- 简历定稿，至少投 5 家公司

## 必交产出物

- [ ] `q-and-a/`：100+ 题自答（含手撕代码）
- [ ] `star-stories/`：3-5 个 STAR 化项目故事
- [ ] `resume-drafts/`：简历定稿（中文 + 英文版）
- [ ] 投递记录：至少 5 家公司

## 每天拆解（W24 一周）

- [ ] [小林 coding 530+ 题](https://xiaolincoding.com/other/ai.html) 全部过一遍
